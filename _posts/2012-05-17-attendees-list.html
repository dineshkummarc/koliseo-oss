List of attendees

To prepare a list of attendees you must click on the Attendees button in your venues list or venue page.

Once in the attendees page, you can select the venue and the date to prepare the list of attendees. The field will display the number of scheduled performances and sold tickets for each date.

Once the venue and date have been selected, hit submit and a list of purchased tickets will be displayed, each displaying the name of the user, the number of tickets and the ticket localizer. You can now print this page and take it to the venue entrance for manual ticket validation.

Manual validation

To validate tickets using a list of attendees, prepare a pen and start checking out tickets. You should verify:

  1. The localizer ID.
  2. The user name.
  3. The number of tickets.

You can choose to print the list of attendees by localizer or by user name.

Did you know? You can validate tickets much faster using Koliseo Checkpoint Charlie for Android. You can just install it on any Android device at the entrance of the venue and start validating in minutes.
