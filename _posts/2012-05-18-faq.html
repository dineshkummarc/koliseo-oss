What does beta mean?

Koliseo is a platform still under development. "Beta" means that we still have wrinkles to iron and it's possible that at some point the application does not behave as expected. We are working hard to avoid this, but sometimes it happens.

How do I create my user account?

Koliseo does not manage user accounts. Instead, you can use any of the user account providers available in the login page. All of them also allow you to create a new user account for free. Koliseo does never have access to your password at any time.

How do I change my avatar?

The avatar (the picture associated to your user account) can be configured at gravatar.com. Just create an account with your user e-mail and upload your new profile picture.
